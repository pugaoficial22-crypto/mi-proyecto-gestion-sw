# Sistema de Gestión de Usuarios

Proyecto de práctica para análisis SAST con SonarQube y GitHub Actions.

## Estructura
- `src/usuario.py` - Modelo de usuario
- `src/database.py` - Capa de datos
- `src/gestor.py` - Lógica de negocio
- `tests/` - Pruebas unitarias

## CI/CD
Este proyecto usa GitHub Actions con SonarQube para análisis estático de seguridad.
